scratchPage

What is it?

A very very simple website which allows you to use jQuery to load you locally stored .js scripts to experiment with. Also, my case study to learn new technologies. This as much as a study journal as it’s a “real” project.

The name says it pretty well:

scratch as in scratchbook for quick notes and Page because it is a page. Duh.

Why?

The idea came forth when I was doing some JavaScript tutorials, and just flat-out hated the web based editors after 2 years on Vim. Also, I like to have my files stored locally for future reference & recycling code.

So, I made this. Sure, it’s very amateurish (and done a million times before), but I learned more about JavaScript & jQuery in one night with this, than on a whole week of tutorials I was doing before. Gotta start somewhere, right?
